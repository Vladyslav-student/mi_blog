## 1. Introducción
En esta práctica he creado y desplegado un sitio web estático utilizando **HTML5, CSS3 y JavaScript**. He utilizado **Git** para el control de versiones (conectando vía **SSH**) y **GitHub Pages** para el alojamiento automático.

## 2. Proceso de Desarrollo
He seguido una metodología incremental con commits atómicos:

- **Fase 1 (Estructura):** Creación de la plantilla base.
- **Fase 2 (Git/GitHub):** Configuración del repositorio y claves SSH en Lubuntu.
- **Fase 3 (Personalización):** Adaptación de contenidos iniciales.
- **Fase 4 (Refactorización Profesional):**
    - Implementación de **Diseño Responsivo** con CSS Grid y Flexbox.
    - Uso de **Variables CSS** para gestión de colores.
    - Desarrollo de un script para cambio de tema (**Modo Claro / Modo Oscuro**).

## 3. Evidencias
El historial de cambios (`git log`) muestra la evolución del proyecto paso a paso.
La web es totalmente funcional y accesible desde el enlace superior.
